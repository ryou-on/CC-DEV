Release Notes

名刺Vault — 使い方

基本操作

  • ＋ボタン → 名刺追加（PDF・JPEG・スマホ写真対応）
  • 「🤖 AIで自動解析」→ Claude Vision OCR でフィールド自動入力
  • 顔写真エリア → 別途顔写真を登録可能
  • カードクリック → 詳細表示・メモ編集
  • ⬇ CSV → Salesforce 取り込み用 CSV 一括出力
  • 検索バー → 名前・会社・メールでリアルタイム絞り込み

⚙️ 設定

  • Cloudflare Worker URL + トークン：Worker 経由（API キー安全）
  • Anthropic API キー：直接呼び出し（個人利用向け）

🔒 セキュリティ

  • Google ログインのみアクセス可能
  • データは Firebase（Firestore / Storage）に保存
