# sound-field-sim（音場シミュレーター）

畳部屋（6/15/30畳）の音場をFDTDで可視化するシミュレーター。
本番: https://cc-dev-ps7.web.app/sound-field-sim/

## バージョン管理ルール（このアプリ固有・厳守）

**過去バージョンをタブで遡れる構成にしている。新バージョンを出すときは必ず以下の手順に従うこと。**

1. `index.html` はタブ付きシェル（iframeコンテナ）。アプリ本体を書かない
2. アプリ本体は `v1/` `v2/` … のようにバージョンディレクトリに `index.html` として置く
3. **過去バージョンのディレクトリは変更・削除しない**（アーカイブとして凍結）
4. 新バージョン追加時:
   - 新しい `vN/index.html` を追加
   - シェル `index.html` 内の `VERSIONS` 配列の**先頭**に1行追加（先頭＝デフォルト表示＝最新）
   - マイナー更新（同じメジャーバージョンの改良）は該当 `vN/` を上書きし、`VERSIONS` のラベルを更新
5. URLハッシュでバージョン指定可: `#v1` `#v2`
