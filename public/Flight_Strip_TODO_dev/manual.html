01概要

FLIGHT STRIP TODO は、航空管制のフライトストリップ（進行スリップ）をモチーフにしたタスク管理ツールです。タスクを「飛行機の便」に見立て、進捗に応じてカラムを横断させながら管理します。

  • Google アカウントでログインするため、複数デバイスで同一データを共有できます
  • タスクは Firestore にリアルタイム同期され、別タブ・別端末の変更が即座に反映されます
  • 開発版は最新機能を先行実装した検証用ビルドです
URL（開発版）： https://cc-dev-ps7.web.app/Flight_Strip_TODO_dev/

02ログイン／ログアウト

  1. アプリを開くとログイン画面が表示される
  2. 「G SIGN IN WITH GOOGLE」ボタンをクリック
  3. Googleアカウントを選択するとアプリが起動する

ヘッダー右上のユーザー名の下にある「SIGN OUT ↩」をクリックするとログアウトします。

05カラム（ステータス）

タスクは6つのカラムのいずれかに属します。

カラム 意味 上限 キー
▶ ACTIVE今まさに取り組んでいるタスク。タイマー計測・全画面モード可11
→ NEXT ACTION次に着手する予定のタスク。①〜⑤番号バー付き52
✈ DELEGATED他者に委任して返答待ちのタスク—3
⏸ HOLDING今すぐ着手しないが保留・待機中のタスク—4
✓ CLEARED完了したタスク。日付別グループ折りたたみ表示—5
✗ LOST中止・破棄したタスク（開発版のみ）—6
開発版追加： CLEARED は日付別グループで折りたたみ可能。LOST カラムで中止タスクを分離管理できます。

06ストリップを作る

  1. ツールバーの「+ FILE STRIP」ボタンをクリック（またはキーボード N）
  2. フォームが展開されるので各フィールドを入力する
  3. 「FILE STRIP」ボタンをクリック（または Enter）で作成
  4. 作成したストリップは NEXT ACTION に追加される
タイトル欄で Enter を押しても作成できます。Esc でフォームを閉じます。

07ストリップを移動する

方法 ①：ドラッグ＆ドロップ

ストリップをつかんで別のカラムへドロップします。カラム内での並べ替えも可能です。

方法 ②：キーボード

キー移動先
1ACTIVE
2NEXT ACTION
3DELEGATED
4HOLDING
5CLEARED
6LOST（開発版）

方法 ③：✓ DONE ボタン

各ストリップ右端の ✓ DONE ボタンで即座に CLEARED へ移動します。CLEARED から ↩ UNDO で ACTIVE に戻ります。

カラム内での並べ替え

ストリップを選択した状態で ↑ / ↓ キーを押すと同カラム内で移動します。

08ストリップを編集・削除する

編集

  • ストリップを選択して E を押す
  • または、ストリップにホバーして鉛筆アイコン ✎ をクリック
開発版追加： タイトルをダブルクリックするとインライン編集モードに入ります。

削除

  • ストリップを選択して D を押す
  • または、ホバーして ✕ ボタンをクリック
注意：削除したストリップは復元できません。

09タイマー

ACTIVE カラムのストリップには作業時間を計測するタイマーが付いています。Firestoreに保存されるため多端末で同期します。

  • 開始：「▶ START」ボタン（またはキーボード T）
  • 停止：「⏹ STOP」ボタン（またはキーボード T）
  • 停止すると作業ログ（timeLogs）に記録され、累計時間が表示される
  • ACTIVE から他カラムへ移動するとタイマーは自動停止

10プロジェクト（シーケンス管理）

開発版機能 — ツールバー左サイドバーの 📋 アイコンから開きます。

複数のタスクを「プロジェクト」としてまとめ、シーケンス（順序）を管理できます。

プロジェクトの作成

  1. サイドバーの「PROJECT」メニューを開く
  2. 「+ 新しいプロジェクト」をクリックしてプロジェクト名と色を入力
  3. タスクを新規作成するか「既存から選ぶ」タブで既存タスクを追加

順番通りモード（デフォルト）

プロジェクトヘッダーの「🔢 順番通り」モードでは、前のタスクが完了するまで次のタスクがボードに表示されません。完了時に次タスクが自動昇格します。

自由順モード

「🔀 自由順」ボタンをクリックで切り替えると、全タスクが同時にボードに表示され、任意の順番で完了できます。完了後は残りの先頭タスクが自動昇格します。

トグル切替：プロジェクトモーダルのヘッダー右の「🔢 順番通り / 🔀 自由順」ボタンを押すだけで即座に切り替わります。

11全画面モード

開発版機能

ACTIVE カラムのヘッダー右の「全画面」ボタンをクリックすると、アクティブタスクに集中するためのフルスクリーンオーバーレイが表示されます。

  • タスクタイトル・チェックリスト・完了ボタンのみを表示するシンプルなレイアウト
  • 右上の ✕ ボタン、または Esc で閉じる
  • チェックリストやルーティンのチェックも全画面から操作可能

12タイムライン

HYBRID / TIMELINE ビュー時に画面上部に表示される時間軸です。各ストリップの「時刻」フィールドに基づいてタスクが配置されます。

表示時間帯の設定

タイムラインヘッダーの「表示時間:」セレクトで開始・終了時刻を変更できます。設定はブラウザに保存されます（デフォルト: 06:00〜22:00）。

ドラッグ操作

  • タスクカードをタイムライン上の別の時間帯にドラッグすると時刻が変更されます
  • タイムライン外のエリアにドロップすると時刻なしになります

現在時刻ライン

赤いバーが現在時刻を示します。表示範囲外の時刻は「時間外 X件」バッジで通知されます。

13繰り返しタスク

開発版機能

編集モーダルの「繰り返し」セクションで繰り返し設定ができます。

  • 毎日 / 毎週（曜日指定） / 毎月（日付指定）
  • タスクを CLEARED にすると、次回分のタスクが自動生成されます
  • 未来の繰り返しタスクはボードに表示されません（当日になると自動表示）

14チェックリスト・ルーティン

チェックリスト

編集モーダルの「チェックリスト」タブでサブタスクを追加できます。

  • 各ストリップ内のチェックボックスをタップ／クリックで完了・未完了を切り替え
  • 全チェック完了してもタスク自体は自動完了しません（手動で DONE ボタンを押す）

ルーティン

開発版機能

毎日のルーティン手順をステップ管理できます。ルーティンのチェックは日次でリセットされます。

16複数選択

開発版機能

複数のストリップを一括で操作できます。

  1. ストリップをクリック後、別のストリップを Shift+クリックで範囲選択
  2. または Ctrl（Mac: ⌘）+クリックで個別に追加選択
  3. ツールバーの一括操作ボタンでステータス変更・削除ができる

18フィードバック送信

開発版機能

ヘッダーの「フィードバック」リンクからフィードバックを送信できます。

  • 不具合報告・機能要望・改善提案など何でも送信できます
  • メールアドレスを入力すると管理者からの返信を受け取れます

19データ管理

ツールバーの「S DATA」メニューから操作できます。

  • JSON エクスポート：全ストリップデータを JSON ファイルとしてダウンロード
  • Markdown エクスポート：タスクリストを Markdown 形式でダウンロード
  • JSON インポート：以前にエクスポートした JSON ファイルを読み込む（既存データに追加）

📋 CONSOLE COPY（デバッグ用）

フッターの「📋 CONSOLE COPY」ボタンで全タスクの JSON データをクリップボードにコピーします。不具合報告時に使用してください。
