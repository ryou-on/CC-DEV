01概要

FLIGHT STRIP TODO は、航空管制のフライトストリップ（進行スリップ）をモチーフにしたタスク管理ツールです。タスクを「飛行機の便」に見立て、進捗状況に応じてカラムを横断させながら管理します。

  • Googleアカウントでログインするため、複数デバイスで同一データを共有できます
  • タスクはFirestoreにリアルタイム同期され、別タブ・別端末の変更が即座に反映されます
  • キーボードショートカットで高速操作できます
URL： https://cc-dev-ps7.web.app/Flight_Strip_TODO/

02ログイン／ログアウト

ログイン

  1. アプリを開くとログイン画面が表示される
  2. 「G SIGN IN WITH GOOGLE」ボタンをクリック
  3. Googleアカウントを選択するとアプリが起動する
ポップアップについて：ブラウザのポップアップブロックが有効だとログインできません。ポップアップを許可してください。

ログアウト

ヘッダー右上のユーザー名の下にある「SIGN OUT ↩」をクリックするとログアウトします。

04フライトストリップ（タスクカード）

各タスクは「フライトストリップ」と呼ばれるカードで表示されます。

◉ FS7842 09:00 URG
企画書を提出する
レポート 締切 仕事

フィールド説明

ストリップID自動採番（FS + 4桁）。変更不可
タイトルタスク名。必須
優先度URG（緊急）/ NRM（通常）/ LOW（低）
カテゴリ仕事 / 家庭 / 学習 / 健康（カスタマイズ可）
時刻タイムラインに表示される予定時刻
タグカンマ区切りで複数入力可
✈ HANDOFF委任先の名前。入力するとオレンジで表示

05カラム（ステータス）

タスクは5つのカラムのいずれかに属します。

カラム 意味 上限 キー
▶ ACTIVE 今まさに取り組んでいるタスク。タイマー計測可能 1 1
→ NEXT ACTION 次に着手する予定のタスク。①〜⑤番号バー付き 5 2
✈ DELEGATED 他者に委任して返答待ちのタスク — 3
⏸ HOLDING 今すぐ着手しないが保留・待機中のタスク — 4
✓ CLEARED 完了したタスク。タイトルに取り消し線が入る — 5
上限について：ACTIVE は1タスクのみ、NEXT ACTION は最大5タスクまで。上限を超えようとするとアラートが表示されます。先に他のタスクを移動してから操作してください。

06ストリップを作る

  1. ツールバーの「+ FILE STRIP」ボタンをクリック（またはキーボード N）
  2. フォームが展開されるので各フィールドを入力する
  3. 「FILE STRIP」ボタンをクリック（または Enter）で作成
  4. 作成したストリップは NEXT ACTION に追加される
タイトル欄で Enter を押しても作成できます。Esc でフォームを閉じます。

07ストリップを移動する

方法 ①：ドラッグ＆ドロップ

ストリップをつかんで別のカラムへドロップします。カラム内での並べ替えも可能です。

方法 ②：キーボード（最速）

  1. J / K でストリップを選択する
  2. 数字キーで移動先を指定する
キー移動先
1ACTIVE
2NEXT ACTION
3DELEGATED
4HOLDING
5CLEARED

方法 ③：ホバーボタン

ストリップにマウスを乗せると右上にボタンが現れます。アイコンボタンをクリックして移動先を選びます。

方法 ④：完了ボタン（✓ DONE）

各ストリップの右端にある ✓ DONE ボタンで即座に CLEARED へ移動します。CLEAREDのストリップで ↩ UNDO をクリックすると ACTIVE に戻ります。

カラム内での並べ替え

ストリップを選択した状態で ↑ / ↓ キーを押すと、同カラム内で上下に移動します。

08ストリップを編集・削除する

編集

  1. ストリップを選択して E を押す
  2. または、ストリップにホバーして鉛筆アイコン ✎ をクリック
  3. 編集モーダルで全フィールドを変更し「SAVE」をクリック

削除

  1. ストリップを選択して D を押す
  2. または、ホバーして ✕ ボタンをクリック
  3. 確認ダイアログが表示されるので「OK」をクリック
注意：削除したストリップは復元できません。

09タイマー

ACTIVE カラムのストリップには、作業時間を計測するタイマーが付いています。 タイマーの状態は Firestoreに保存されるため、スマートフォンとPCで同時に表示が同期します。

  • 開始：ストリップ内の「▶ START」ボタンをクリック（またはキーボード T）
  • 停止：「⏹ STOP」ボタンをクリック（またはキーボード T）
  • 停止すると作業ログ（timeLogs）に記録され、累計時間が表示される
  • タスクを ACTIVE から別カラムへ移動すると、タイマーは自動停止する
多端末同期：同じアカウントでPCとスマートフォンを同時に開いていると、どちらでタイマーを開始・停止しても両方に即座に反映されます。

10フィルター

ツールバーのセレクトボックスで表示を絞り込めます。

  • ALL PRI / URG / NRM / LOW：優先度でフィルター（キーボード F で循環）
  • ALL CAT / 仕事 / 家庭 / 学習 / 健康：カテゴリでフィルター
フィルターはすべてのカラムに同時に適用されます。

11カテゴリ編集

デフォルトカテゴリ（仕事・家庭・学習・健康）はカスタマイズできます。

  1. ツールバーの「CAT ✎」ボタンをクリック
  2. カテゴリ名と色を変更する
  3. 「+ 追加」で新しいカテゴリを追加できる
  4. 右端の「✕」でカテゴリを削除（最低1件は必要）
  5. 「SAVE」をクリックして保存（Firestoreに保存 → 全端末に同期）

12テーマ切替

ツールバーのテーマボタンで背景の配色を変更できます。

DARK LIGHT MEDIUM COLOR COLORFUL

設定はブラウザに保存されるため、次回アクセス時も維持されます。

13文字サイズ切替

ツールバーの「A小」ボタンをクリックするたびに、ストリップのタイトル文字サイズが切り替わります。

A小  →  A中  →  A大  →  A小  （循環）

視力や画面サイズに合わせて調整してください。

14ビューモード（PC のみ）

PC表示時、ツールバーのボタンで3種類のビューを切り替えられます。

HYBRID タイムライン + カンバンを同時表示（デフォルト）
KANBAN カンバンボードのみ表示
TIMELINE タイムライン + カンバンを表示（HYBRIDと同様）

タイムライン（表示時間帯カスタマイズ可）

各ストリップの「時刻」フィールドに基づき、時間軸上に表示されます。現在時刻は赤いラインで表示されます。タイムラインヘッダーの「表示時間:」セレクトで開始・終了時刻を自由に設定できます（設定はブラウザに保存されます）。

15モバイル操作

タッチ操作でのドラッグ＆ドロップ

  1. ストリップを長押しせずにそのままスワイプ（タッチ開始後に移動）
  2. 半透明のゴーストカードが表示され、指を離した場所のカラムに移動する
ページスクロールとドラッグが競合する場合は、ストリップ上ではなく少し外側をスクロールしてください。

タブ切替

ACTIVE の下にタブが表示されます。タップで展開・再タップで収納できます。

  • NEXT — 次のアクション
  • DELEGATED — 委任中
  • CLEARED — 完了済み

HOLDING はタブの下に常時表示されます。

16データ管理

ツールバーの「DATA」メニューから操作できます。

  • JSON エクスポート：全ストリップデータを JSON ファイルとしてダウンロード
  • Markdown エクスポート：タスクリストをMarkdown形式でダウンロード
  • JSON インポート：以前にエクスポートした JSON ファイルを読み込む
インポート時の注意：インポートするとFirestoreの既存データに追加されます。上書きではありません。

📋 CONSOLE COPY（デバッグ用）

フッターの「📋 CONSOLE COPY」ボタンをクリックすると、全タスクのJSONデータをクリップボードにコピーします。不具合報告時に使用してください。
