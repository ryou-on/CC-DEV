📖 使い方
⚙ 初期設定
  1. 右上の「⚙ 設定」をクリック
  2. Claude API キー（sk-ant-...）を入力して保存
  3. ユーザーIDを設定（複数デバイスで同じデータを使う場合に統一する）
📸 手段①：スクリーンショットOCR
  1. note.com のダッシュボードをスクリーンショット（複数枚可）
  2. 画像をドラッグ&ドロップ or クリックして選択
  3. 「まとめてAI解析」→ 数値を確認して保存
🔖 手段②：Chrome自動取得
  1. 「Chrome自動取得」タブ → リンクを右クリックして「ブックマークに追加」
  2. note.com にログインし、統計ページを開く
  3. ブックマークをクリック → データが自動送信される
📊 データの見方
  • 週次トレンドグラフで指標を切替表示
  • サマリーカードで前週比を確認
  • 「CSVエクスポート」でスプレッドシートに活用

🔖 Chrome自動取得の使い方
  1. 下のボタンを右クリック →「ブックマークに追加」
    （または長押しドラッグでブックマークバーへ）
  2. note.comのアクセス状況ページを開く（週表示）
    （ https://note.com/sitesettings/stats ）
  3. 追加したブックマークをクリック
  4. データが自動的にこのアプリに取り込まれます
