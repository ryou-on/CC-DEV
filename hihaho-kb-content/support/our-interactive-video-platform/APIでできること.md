インタラクティブ動画のデータを別のシステムに送りたい、あるいは大量の動画を自動でアップロードしたいと思ったことはありませんか？または、動画内で視聴者が行った選択に応じて、あなたのWebサイトを反応させたい、ということは？こうした機能をはじめ、さらに多くのことが、私たちのAPIを利用することで実現できます。APIとは、あるソフトウェア製品と別のソフトウェア製品の間でデータのやり取りを可能にするプログラミングコードの集まりです。

私たちは3つのAPIを提供しています。以下でそれぞれを簡単にご紹介します。

---

## 作成・アップロード用API

作成・アップロード用APIは、動画を自動的にアップロードしたり、インタラクティブ化したりするために使用します。このAPIは、たとえば次のような場合に使います。

* インタラクティブ動画を自動生成したい
* 定型フォーマットの大量の動画をインタラクティブ化したい
* 動画を自動でアップロードしたい

**[APIドキュメントを見る](https://api-docs.hihaho.com/)**

---

## データAPI

視聴者があなたの動画を見ると、多くのデータが得られます。このデータは、hihahoにログインした状態で統計ページから確認できます。このAPIを使うと、これらのデータを自社のアプリケーション（アドオン）に統合できます。このAPIは、たとえば次のような場合に使います。

* メールシステムのデータとhihahoのデータを1つのビューに統合したダッシュボードを作成したい
* インタラクティブなクイズやテストで、複数のプレイヤーのスコアを表示するスコアボードを作成したい

---

## JavaScript API

自社のプラットフォームとインタラクティブ動画をリアルタイムに連携させたいですか？それを可能にするのが JavaScript API です。

JavaScript API は、たとえば次のような場合に使います。

* インタラクティブ動画内でのクリックに応じて、Webサイト上の情報を変化させたい

**[JavaScript APIドキュメントをダウンロード](https://www.hihaho.com/wp-content/uploads/2022/03/Hihaho-JS-API-trigger-values-general-2021-11-29.pdf)**

---

これらのAPIを使えば、自動化の仕組みなしでは実現できないことが可能になります。
