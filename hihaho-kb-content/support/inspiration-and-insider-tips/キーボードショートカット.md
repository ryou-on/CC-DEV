**キーボードショートカット**を使うと、インタラクティブ動画の編集時間を大幅に節約できます。同様に、動画の視聴者もインタラクティブ動画を見ながらキーボード操作を使うことができます。

## 基本のキーボード操作

| キー | 動作 |
|-----|--------|
| スペースキー | 動画の再生 / 一時停止 |
| Shift | インタラクションのサイズ変更時に縦横比を維持 |
| ▲ | 音量を5%上げる |
| ▼ | 音量を5%下げる |
| F | フルスクリーンモード（プレイヤー/プレビューのみ。編集モードでは不可） |
| Escape | フルスクリーンモードを終了（プレイヤー/プレビューのみ） |
| 左右矢印キー | タイムラインを1秒移動 |
| 左右矢印キー + Shift | タイムラインを5秒移動 |
| 左右矢印キー + Alt | タイムラインを10ミリ秒移動 |
| 左右矢印キー + Alt + Ctrl | タイムラインを100ミリ秒移動 |
| Ctrl + s | 保存（動画設定内） |

## キーボード操作 — ポップアップメニュー

| キー | 動作 |
|-----|--------|
| Tab または Ctrl + スペース | メニュー項目を追加 |
| Shift + Tab | メニュー項目を削除 |

## キーボード操作 — 質問

質問を編集しているときは、以下のショートカットキーが使えます。

| キー | 動作 |
|-----|--------|
| Tab | 回答を追加 |
| Shift + Tab | 回答を削除 |

質問が画面に表示されたとき、視聴者はキーボードで回答できます。操作は以下のとおりです。

| キー | 動作 |
|-----|--------|
| Enter | 質問の回答を送信 |
| Ctrl + Enter | 質問の回答を送信 |
| Shift + Tab または ↑ | 前の回答を選択 |
| Tab または ↓ | 次の回答を選択 |
| スペースキー | チェックボックスにチェックを入れる |
