執筆: Melanie（2021年11月23日）

Pause Screen（一時停止画面）はさまざまな方法で活用できることをご存知ですか？この記事では、次の活用法を紹介します:

1. ひと息つく時間を作る
2. 自作のメニューや質問のために動画を一時停止する
3. 補足のテキスト情報を提示する
4. 内省を促す問いかけを作る

---

## 1. ひと息つく時間を作る

視聴者に続きを見る前に少し時間を与えたいときは、Pause Screen を「Click to continue（クリックして続行）」ボタンに変えることができます。

たとえば、視聴者がこの図のすべての部分をじっくり確認するのに時間が必要な場合などです。ボタンをクリックすると、動画の一時停止が解除されます。

## 2. 自作のメニューや質問のために動画を一時停止する

複数のインタラクションを組み合わせて独自の質問やメニューを作るときは、通常、動画を一時停止させたいはずです。

その場合は、[非表示の Pause Screen（hidden Pause Screen）](https://support.hihaho.com/en/articles/5755415-hidden-pause-screen)を追加する必要があります。

![パーソナライズされたメニューと質問の例](https://hihaho-d6f71b986460.intercom-attachments-1.com/i/o/412148131/8a847d088d91758965abbb5f/Make-your-own-personalized-menus-and-questions_1.png?expires=1783476000&signature=c8f69a2b6ac5e32bebd88edb9de0329abf7c9c419ed1555e1704168950ef26a0&req=cCElF812nIJeFb4f3HP0gIrxa1HcAnmdntFc%2BQJVNcctTdfmBJ%2Fku%2BLEVUUC%0AYJa6Pg9IofJ4xaVlVQ%3D%3D%0A)

## 3. 補足のテキスト情報を提示する

Pause Screen は、動画の要約などの補足テキスト情報を提示するのにも使えます。動画内に映っている内容についてコメントを加えることもできます。フルスクリーンでも、小さく表示しても、半透明にしても構いません。

## 4. 内省を促す問いかけを作る

同じ方法で、内省を促す問いかけ（リフレクション用の質問）を作ることもできます。Question（質問）インタラクションとは異なり、視聴者は回答を入力する必要はありません。ただ考えるだけでよいのです。以下の例では、フルスクリーンの Pause Screen を追加しています。
