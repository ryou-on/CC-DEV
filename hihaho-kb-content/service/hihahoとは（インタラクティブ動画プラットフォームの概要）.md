hihahoは、ビジネスをスマートに変革する次世代動画プラットフォームです。製品コンセプトは「**驚くほど簡単なのに高機能な**」インタラクティブ動画プラットフォーム。ラーニング分野のインタラクティブ動画制作ツールとして No.1 の評価を得ています。

## 必要なものは動画素材とブラウザだけ

ソフトウェアのインストールは不要。手持ちの動画素材をアップロード（または動画プラットフォームから選択）し、ブラウザ上のエディタで質問・一時停止・ジャンプ・フォーム・メニューなどのインタラクションをタイムラインに配置するだけで、インタラクティブ動画が完成します。

## インタラクティブ動画とは

動画素材の上に「インタラクティブ層」を加えることで、さまざまな機能が追加された**双方向性のある動画**です。追加できる要素の例：

- 目次 / 問題 / 選択肢 / アンケート
- 予約 / フォーム
- リンク機能 / ショッピング / リード獲得 / API連携

## 従来の動画との違い

| 従来の動画 | hihahoを使った動画 |
|---|---|
| 受動的視聴 | 能動的視聴・触れて選択 |
| 行動の終着点（見て終わり） | 申し込み・購入・予約へ直接行動 |
| 低い学習効率 | 回答採点・自己学習・自己解決 |
| 広告あり（YouTube等） | 広告なし |
| 低いブランド体験 | 高いブランド体験・疑似体験・検討選択 |

ターゲットの興味を直接的な行動につなげる「仕組み化」をサポートします。

## いつでも、どこでも、どんなデバイスでも

PC、タブレット、スマホ、デジタルサイネージなど、あらゆる環境で視聴できます。
