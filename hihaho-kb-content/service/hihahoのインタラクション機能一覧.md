hihahoには16種類のインタラクション機能があり、動画の上に自由に配置できます。充実の機能に加え、毎月機能アップデートが継続中です。

## 16種類のインタラクション機能

| 機能 | 説明 |
|---|---|
| テキスト | 動画にテキストを追加。アクション機能で動画内の好きな時間や外部サイトへジャンプさせられる |
| 画像 | ロゴ、バナー、インフォグラフィックなどの画像を追加 |
| ポップアップメニュー | 動画内の特定時間やURLへのジャンプを提供し、視聴者の利便性を高める |
| 目次 | 動画に目次を追加して利便性を高める |
| ホットスポット | 注目ポイントの表示、説明、情報リンクをする最もシンプルな方法 |
| スクロールテキスト | 流れるテキストでニュースフィード表示を実装 |
| 透明リンク | 動画に含まれる要素をクリック可能な領域に。動きも自由に設定可能 |
| フォーム | 情報入力フォームを動画に追加。データはメール送信やCRMと連携可能 |
| 動画リンク | シナリオ分岐動画が簡単に作れる |
| ハイライト | 動画の一部をハイライト表示し、注目を一か所に集める |
| ズーム | ダイナミックにある一点を拡大表示 |
| 評価 | 動画やコンテンツの評価を収集 |
| 質問 | 動画に質問を追加してエンゲージを高める |
| ジャンプ | 任意の時間に動画をジャンプ。繰り返し再生するループとしても利用可能 |
| 一時停止 | 視聴者の注意を獲得するのに有効 |
| サウンド | 動画に効果音を追加 |

※利用可能な機能はプランにより異なります。
