      📖 使い方
      
    

    

      

        
STEP 1

        
YouTubeの動画ページを開く

        
動画を再生しているタブでこのアイコン（▶️）をクリック

      

      

        
STEP 2

        
「生成 & デプロイ」を押す

        
スラッグ（URL末尾）を確認 → ボタンを押すと文字起こし・AI分析・GitHubデプロイが自動で実行されます

      

      

        
STEP 3

        
hihaho に設定

        
完了後のタブURLをコピー → hihaho studio → インタラクション → iFrame → 全画面設定で貼り付け → 公開

      

      

        ⚠ 必要な設定（⚙ ボタン）
        
    
          
  • Claude API Key（必須）
    • 
          
  • GitHub Token / Repo（必須・Contents write権限）
    • 
          
  • OpenAI API Key（任意・字幕なし動画向け）
    • 
          
  • hihaho API情報（任意・動画を自動作成）
    •
